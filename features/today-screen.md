# Today screen

Flow: .maestro/today-screen.yaml
Platforms: ios, android, web

Creates a family event for today, a pinned note, and an active project.
Then opens Today and verifies all three appear before deleting them.

## Acceptance criteria
- A family event created for today appears in the Today unified list.  (visual)
- A pinned note appears in the PinnedNotesCarousel on Today.           (visual)
- An active project appears in the ActiveProjectsCarousel on Today.   (visual)
